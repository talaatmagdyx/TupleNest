# Contributing to TupleNest

TupleNest is an MIT-licensed open-source project.

- Discuss significant changes in an issue first.
- Rust: `cargo fmt`, `cargo clippy -- -D warnings`, `cargo test` must pass.
- Never log or commit credentials; secrets live only in the OS keychain.
- Every driver change must pass the driver contract tests.
- See docs/phase-0-plan.md and docs/phase-1-plan.md for the active backlog.
